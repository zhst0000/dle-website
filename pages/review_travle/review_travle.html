Travle Review

Published: 7/21/2025

Read Length: 5-10 min

Travle is a geography game where the player is tasked with creating a path from one country to another that passes through as few other nations as possible. The game is run by TeufTeuf who also produce many other heavy hitters in the geography daily game space such as GeoGrid, and Worldle.

[CAPTION]

To start a game a player is given two countries that will each represent either the start or the end of that day’s trip. The object of the game is to identify the path between these countries that minimizes the total number of international border crossings. A player who identifies all of the countries that one would need to pass through in order, and without guessing any extraneous countries will receive a “perfect” score. If the player gets the wrong order then they receive an imperfect score of “+0”; each unnecessary country will worsen your score by an additional point (e.g. “+1”, “+2”, etc.)

(A completed game where I guessed the correct countries, but in the wrong order and the results page showing my score “+0” and how that compares to everyone else who played today)

Beyond just the base game, Travle has several additional ways to engage with its core mechanic. Firstly, the site offers weekly challenges where the rules of the game are altered. For example, the rules could require the player to travel between the two countries along the coast, or using borders from another period in history. Next, instead of forming a path between countries you can play on maps based on subdivisions within countries or regions: building paths between U.S. states, British counties, or Japanese prefectures to name a few. Finally, travle offers a practice mode that allows the player to play the game between any two countries of their choosing or generate random games of a set length.

(a completed weekly challenge)

This game is reasonably welcoming to people of various knowledge levels as you can request several hints to help if you get stuck. These hints range in strength from showing the outline of the next country all the way to showing the outline and starting letter of all countries. However if you can't come up with the name of a country with these hints it may be impossible to complete that day's puzzle provided there is no other route to take. Another aspect that may make this game tough for a newcomer is knowing which countries are actually part of the dataset and knowing when you can “jump” over bodies of water to connect enclaves and when you cannot. For example Spain does border both Morocco and France however the shortest allowable path between these countries does not pass through Spain because there is no land border between Spain’s north-african enclaves and the mainland. However, the game does allow the UK as part of the path between Ireland and France despite there not being a land border between Northern Ireland and Great Britain. Though, this and similar lapses in consistency are rarely relevant.

As a lover of geography games I find Travle to be among the best. Where most geography games focus on identifying places based on images or memorizing details about a nation like its flag or population, Travle’s insistence on making the world map itself the center of its gameplay loop make it fairly unique in the genre. My biggest concern with this game is its longevity. I've played over a hundred of these puzzles and after a while you realize that there are only so many different interesting paths to take. Any two puzzles with similar starting and ending points will have mostly the same intermediate countries. Still this game is a fun way to stay in practice in remembering the minutiae of world borders.

Travle is a polished, and feature-dense game whose unique approach to its content sets it apart from other games in this category
